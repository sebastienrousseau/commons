# Commons

Shared Rust utilities and common patterns for the Sebastien Rousseau ecosystem.

## Overview

Commons provides reusable components, traits, and utilities used across multiple Rust projects in the ecosystem.

## Modules

- `error` - Common error types and handling
- `config` - Configuration management utilities
- `logging` - Structured logging setup
- `testing` - Test utilities and fixtures

## Installation

```toml
[dependencies]
commons = { git = "https://github.com/sebastienrousseau/commons" }
```
